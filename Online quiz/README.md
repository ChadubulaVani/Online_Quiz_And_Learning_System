# 🎓 Online Quiz & Learning System

An interactive web-based platform developed using the MEAN stack (MongoDB, Express.js, Angular, Node.js) that enables students to learn through structured courses, watch multimedia content, and assess their knowledge through quizzes.

---

## 🚀 Project Overview

The Online Quiz & Learning System is designed to provide a smart and user-friendly learning environment. It allows users to register, access categorized courses, watch video lectures, and take quizzes with real-time evaluation.

This system helps students improve their knowledge while providing administrators with tools to manage content efficiently.

---

## 🎯 Key Features

### 👨‍🎓 Student Features
- User Registration & Login (JWT Authentication)
- Browse Courses by Category
- Watch Video Lectures
- Attempt Quizzes (MCQs)
- Instant Score Calculation
- Performance Tracking

### 👨‍💼 Admin Features
- Manage Users
- Add / Update / Delete Courses
- Upload Learning Content (Videos/Notes)
- Create and Manage Quizzes
- Monitor Student Performance

---

## 🧠 Advanced Features (Enhancements)

## 🏗️ System Architecture

Frontend → Angular  
Backend → Node.js + Express.js  
Database → MongoDB Atlas  

(Three-tier architecture ensuring scalability and performance)

---

## 🛠️ Tech Stack

- **Frontend:** Angular, TypeScript, HTML, CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB Atlas  
- **Authentication:** JWT (JSON Web Tokens)  
- **Version Control:** Git & GitHub
